Entendendo Comunicação Client x Server

A comunicação entre Client e Server é a base do funcionamento da Web e de diversas aplicações diditais, onde o client solicita algo e o server processa as requisições e retorna respostas.

O que são Clients

Clients são dispositivos ou softwares que consomem serviços de um servidor.
Clients → Internet (Protocolo HTTP) → ← Servers

O que são Servers

Servers são responsáveis por processar e armazenar dados, atendendo o que os clientes querem.

Tipos de Servers:

  • Arquivos
  • Segurança (Firewall)
  • Streaming
  • E-mail
  • Web

Os servidores geralmente rodam em nuvens ou em infraestruturas locais.

Linguagens de Programação

Linguagem escrita e fomral que específica um conjunto de instruções e regras usadas para gerar programas.

  • Server-side: são executadas no servidor
  • Client-side: São executadas no client
